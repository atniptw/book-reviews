# The Pragmatic Programmer

**Author:** David Thomas, Andrew Hunt

**Rating:** 5/5

**Review Date:** 2026-01-25

---

## Review

This book is a must-read for any software developer. It provides practical advice and timeless wisdom that applies across programming languages and paradigms. The authors present their ideas in an accessible way with real-world examples that make the concepts easy to understand and apply.

## What I Liked

- Clear and concise writing style
- Practical tips that can be applied immediately
- Timeless advice that remains relevant decades after publication
- Great use of analogies and metaphors
- Covers both technical and professional development topics

## What I Disliked

- Some examples use older technologies (though the principles still apply)
- Could use more modern code examples in newer editions

## Additional Notes

Key takeaways:
- Don't repeat yourself (DRY principle)
- Fix broken windows (don't let bad code accumulate)
- Be a catalyst for change
- Use tracer bullets to find the target
- Program deliberately
